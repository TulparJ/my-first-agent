# My First AI Agent
A tool-using AI chatbot built with the Anthropic Claude SDK.

## Features
- 🧮 Calculator
- Weather
- Image Generation
- Write Files
- Run Code
- 🧠 Memory

## Installation
pip3 install -r requirements.txt

## Setup
export ANTHROPIC_API_KEY="your-key"
export HF_API_KEY="your-key"

## Run
python3 chatbot.py
